# Projeto Flexbox

## Descrição
Este projeto é uma página de demonstração prática dos principais recursos do **CSS Flexbox**. Ele apresenta exemplos de diferentes propriedades e comportamentos, permitindo visualizar como cada configuração afeta a disposição dos elementos na tela.

## Estrutura de Arquivos
- `index.html`: arquivo principal contendo a estrutura da página e os exemplos de Flexbox.
- `css/styles.css`: folha de estilos responsável pela aplicação das propriedades de Flexbox e estilos visuais.

## Conteúdo Demonstrado
O projeto inclui exemplos das seguintes propriedades e conceitos do Flexbox:

1. **Flex**: uso básico de `display: flex`.
2. **Flex Direction**: organização em coluna (`flex-direction: column`).
3. **Flex Wrap**: quebra de linha automática (`flex-wrap: wrap`).
4. **Justify Content**: alinhamento horizontal (`center`, `space-around`, `space-between`, `end`).
5. **Align Items**: alinhamento vertical (`center`, `flex-end`, `stretch`).
6. **Gap**: espaçamento entre elementos.
7. **Order**: alteração da ordem dos elementos.
8. **Flex Grow**: crescimento proporcional dos itens.
9. **Flex Basis**: definição de tamanho base dos itens.
10. **Flex Shrink**: redução proporcional dos itens.
11. **Flex Shorthand**: uso da propriedade abreviada `flex`.
12. **Align Self**: alinhamento individual de elementos dentro do container.

## Tecnologias Utilizadas
- HTML5
- CSS3 (Flexbox)

## Como Executar
1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Explore os diferentes exemplos para entender o comportamento das propriedades do Flexbox.

## Objetivo
O projeto serve como material de estudo e referência para desenvolvedores que desejam aprender ou revisar os conceitos fundamentais do Flexbox.
