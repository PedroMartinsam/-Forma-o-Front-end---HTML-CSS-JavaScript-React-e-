# Projeto HTML Semântico

## Descrição
Este projeto demonstra o uso de **tags semânticas do HTML5**, organizando o conteúdo de forma clara e estruturada.  
O objetivo é praticar boas práticas de marcação semântica, melhorando a acessibilidade e a compreensão do código.

## Estrutura de Arquivos
- `index.html`: arquivo principal contendo a estrutura semântica da página.
- `img/`: pasta de imagens utilizadas no projeto (exemplo: `lambo.jpg`).

## Conteúdo Demonstrado
O projeto utiliza diversas tags semânticas do HTML5:

- **`<header>`**: cabeçalho do site com título e navegação.
- **`<nav>`**: menu de navegação com links principais.
- **`<main>`**: conteúdo principal da página.
- **`<section>`**: agrupamento de conteúdos relacionados (produtos, diferenciais, posts).
- **`<article>`**: posts independentes do blog.
- **`<aside>`**: barra lateral com categorias.
- **`<mark>`**: destaque de conteúdo importante dentro de um texto.
- **`<figure>` e `<figcaption>`**: imagem com legenda descritiva.
- **`<footer>`**: rodapé com informações de contato e créditos.

## Tecnologias Utilizadas
- HTML5

## Como Executar
1. Clone ou baixe este repositório.
2. Certifique-se de que a pasta `img/` contém as imagens necessárias.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
4. Explore a página para visualizar o uso das tags semânticas.

## Objetivo
O projeto serve como material de estudo e referência para desenvolvedores que desejam aprender ou revisar o uso de **HTML semântico** em páginas web.
