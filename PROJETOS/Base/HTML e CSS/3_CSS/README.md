# 3_CSS

## Descrição
Projeto voltado ao aprendizado e prática de **CSS**, demonstrando diferentes formas de aplicar estilos: **inline, internal e external CSS**. Inclui exemplos de cores, fontes, backgrounds, box model, posicionamento, listas, títulos e outros elementos visuais importantes.

O projeto traz vários exercícios e desafios que permitem experimentar conceitos de CSS de forma prática.

---

## Arquivos do projeto

### 1. `index.html`
- Demonstrações de:
  - Inline CSS (`style` diretamente nas tags)
  - Internal CSS (`<style>` dentro do `<head>`)
  - External CSS (vários arquivos separados: `styles.css`, `lists.css`, `titles.css`)
  - Estilização de títulos (`<h1>`, `<h2>`, `<h3>`, `<h4>`)  
  - Parágrafos (`<p>`) e manipulação de cores, fontes e tamanhos  
  - Divs e box model  
  - Listas ordenadas e não ordenadas  
  - Posicionamento (`static`, `relative`, `absolute`, `fixed`, `sticky`)  
  - Z-index, backgrounds, imagens de fundo, opacidade e efeitos  

### 2. `css/styles.css`
- Contém regras gerais e classes para:
  - Cores (nome, hex, rgb, rgba, hsl)  
  - Backgrounds e imagens  
  - Padding, margin, border, border-radius  
  - Box model completo e desafios de layout  
  - Fontes, text-transform, text-decoration, letter-spacing, font-weight, font-style  
  - Posicionamento e display de elementos  
  - Efeitos de visibilidade e z-index  

### 3. `css/lists.css`
- Estilização de listas (`ul` e `li`):
  - Bordas, padding e cores

### 4. `css/titles.css`
- Estilização de títulos (`h4`):
  - Background color e cor da fonte

---

## Tecnologias utilizadas
- HTML5
- CSS3 (inline, internal e external)
- Prática de boas práticas de organização de CSS

---

## Como rodar localmente
1. Clone este repositório ou faça o download da pasta do projeto.  
2. Abra o arquivo `index.html` no seu navegador para visualizar todas as demonstrações.

---

## Aprendizados
- Diferentes formas de aplicar CSS (inline, internal, external)  
- Uso de cores (nome, HEX, RGB, RGBA, HSL)  
- Manipulação de backgrounds, imagens, opacidade e posicionamento  
- Box model completo: margin, padding, border, border-radius  
- Posicionamento avançado: static, relative, absolute, fixed, sticky e z-index  
- Estilização de textos, fontes, listas e títulos  
- Organização de arquivos CSS e boas práticas de modularização
