
# 8_HDC_HOST - Site de Hospedagem

## Descrição
Projeto de site institucional para uma empresa de **hospedagem de sites**, com HTML e CSS.  
O objetivo é criar um layout completo com **navbar, banner, serviços, planos, busca de domínio e formulário de contato**, demonstrando organização visual, responsividade e estilo moderno.

O site permite visualizar serviços, comparar planos, procurar domínios e enviar mensagens diretamente pelo formulário de contato.

---

## Arquivos do projeto

### 1. `index.html`
Estrutura do site:

- Navbar com links para Home, Preços, Contato e botão de login  
- Banner principal com título e descrição  
- Seção de serviços com ícones e descrição  
- Seção de planos de hospedagem com preços, recursos e destaque para plano recomendado  
- Seção de busca de domínios com input e botão  
- Formulário de contato com inputs de texto, e-mail, textarea e botão de envio  
- Footer com informações da empresa  

### 2. `css/styles.css`
Estilização:

- Layout moderno com cores sóbrias e elementos destacados  
- Navbar fixa com hover nos links e botão destacado  
- Banner com imagem de fundo, texto centralizado e sombras  
- Seções organizadas com **Flexbox** para alinhamento horizontal de cards  
- Planos destacados visualmente, incluindo plano recomendado  
- Formulário de contato e busca de domínios com inputs estilizados e botões chamativos  
- Responsividade implementada com media queries para telas de **desktop, tablet e mobile**  

---

## Tecnologias utilizadas
- HTML5  
- CSS3  
- Font Awesome (ícones)  

---

## Como rodar localmente
1. Clone este repositório ou faça download da pasta do projeto.  
2. Abra `index.html` no navegador.  
3. Navegue pelas seções e teste a responsividade redimensionando a tela.
