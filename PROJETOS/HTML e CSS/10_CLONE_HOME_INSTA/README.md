# Clone da Página de Login do Instagram

## Descrição
Este projeto é uma recriação da página de login do **Instagram**, utilizando apenas **HTML** e **CSS**.  
O objetivo é praticar conceitos de layout responsivo, estilização e organização de componentes visuais.

## Estrutura de Arquivos
- `index.html`: estrutura principal da página.
- `css/styles.css`: folha de estilos responsável pela aparência e responsividade.
- `img/`: pasta contendo imagens utilizadas (logo, banner, ícones e botões de download).

## Funcionalidades
- Formulário de login com campos de usuário e senha.
- Botão de login estilizado.
- Separador com opção de login via Facebook.
- Link para recuperação de senha.
- Caixa de registro para novos usuários.
- Área para download do aplicativo (Apple Store e Google Play).
- Rodapé com links institucionais.

## Responsividade
- Layout adaptado para dispositivos móveis com largura máxima de 450px.
- Ocultação do banner lateral em telas pequenas.
- Ajustes de largura e remoção de bordas para melhor visualização em celulares.

## Tecnologias Utilizadas
- HTML5
- CSS3 (Flexbox, Media Queries)

## Como Executar
1. Clone ou baixe este repositório.
2. Certifique-se de que a pasta `img/` contém as imagens necessárias.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
4. Explore a página e teste em diferentes tamanhos de tela para visualizar a responsividade.

## Objetivo
O projeto serve como exercício prático de front-end, reforçando conceitos de **HTML semântico**, **CSS responsivo** e **design inspirado em aplicações reais**.
