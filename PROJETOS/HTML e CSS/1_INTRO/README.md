
# 1_INTRO

## Descrição
Projeto simples de introdução ao **HTML e CSS**, com um título demonstrando a estrutura básica de uma página web.

## Tecnologias utilizadas
- HTML5
- CSS3 (se aplicável)

## Funcionalidades
- Exibe um título na página (`<h1>Este é um título</h1>`)

## Como rodar localmente
1. Clone este repositório ou faça o download da pasta do projeto.
2. Abra o arquivo `index.html` no seu navegador.

## Aprendizados
- Estrutura básica de um documento HTML.
- Uso de tags de título (`<h1>` a `<h6>`).
- Noções iniciais de organização de arquivos para projetos web.
