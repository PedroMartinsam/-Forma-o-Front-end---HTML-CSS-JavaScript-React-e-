
# 2_HTML

## Descrição
Este projeto é voltado ao **aprendizado dos fundamentos do HTML**, incluindo títulos, parágrafos, links, imagens, listas, tabelas, divs e comentários. Contém três arquivos principais que mostram diferentes exercícios e desafios de HTML.

---

## Arquivos do projeto

### 1. `index.html`
- Demonstrações de:
  - Títulos (`<h1>`, `<h2>`, `<h3>`)  
  - Parágrafos (`<p>`) e tags sem conteúdo (`<br>`, `<hr>`)  
  - Comentários HTML  
  - Links (`<a>`), incluindo abrir em nova aba  
  - Imagens (`<img>`) com e sem atributo `alt`  
  - Listas não ordenadas (`<ul>`) e ordenadas (`<ol>`)  
  - Tabelas (`<table>`)  
  - Estrutura com `<div>` e links internos  

### 2. `estrutura.html`
- Modelo básico de estrutura HTML5, incluindo:
  - `<!DOCTYPE html>`  
  - `<html lang="en">`  
  - `<head>` com `meta charset`, `viewport` e título  
  - `<body>` vazio para criar novas páginas  

### 3. `desafio.html`
- Pequeno desafio de HTML:
  - Título e parágrafo explicativo  
  - Imagem com atributo `alt`  
  - Lista de itens relacionados à imagem  

---

## Tecnologias utilizadas
- HTML5
- CSS3 (se aplicável, inline ou em arquivos externos)

---

## Como rodar localmente
1. Clone este repositório ou faça o download da pasta do projeto.  
2. Abra qualquer um dos arquivos `.html` em seu navegador.

---

## Aprendizados
- Estrutura básica de um documento HTML  
- Uso de tags de título, parágrafo, lista, tabela e div  
- Inserção de imagens com atributo `alt`  
- Links internos e externos  
- Organização de arquivos para projetos web iniciais
