# Projeto Blog Semântico

## Descrição
Este projeto é uma página de blog desenvolvida com **HTML5 semântico** e **CSS3**, organizada em estrutura clara e acessível.  
O objetivo é praticar boas práticas de marcação semântica e estilização, criando um layout responsivo com posts, barra lateral e rodapé.

## Estrutura de Arquivos
- `index.html`: arquivo principal contendo a estrutura semântica da página.
- `css/styles.css`: folha de estilos responsável pela aparência e responsividade.
- `img/`: pasta com imagens utilizadas nos posts.

## Conteúdo Demonstrado
O projeto utiliza diversas tags semânticas do HTML5:

- **`<nav>`**: barra de navegação principal.
- **`<main>`**: área principal com os posts.
- **`<article>`**: cada post do blog.
- **`<aside>`**: barra lateral com busca, categorias e tags.
- **`<section>`**: agrupamento de conteúdos relacionados dentro da barra lateral.
- **`<footer>`**: rodapé com informações adicionais.

## Funcionalidades
- Exibição de posts com título, descrição, autor e imagem ilustrativa.
- Barra lateral com:
  - Campo de busca.
  - Lista de categorias.
  - Conjunto de tags.
- Layout responsivo que se adapta a telas menores (até 450px).

## Tecnologias Utilizadas
- HTML5 (semântico)
- CSS3 (Flexbox, Media Queries)

## Como Executar
1. Clone ou baixe este repositório.
2. Certifique-se de que a pasta `img/` contém as imagens necessárias.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
4. Explore a página e teste em diferentes tamanhos de tela para visualizar a responsividade.

## Objetivo
O projeto serve como exercício prático de front-end, reforçando conceitos de **HTML semântico**, **CSS responsivo** e **design de blogs**.
