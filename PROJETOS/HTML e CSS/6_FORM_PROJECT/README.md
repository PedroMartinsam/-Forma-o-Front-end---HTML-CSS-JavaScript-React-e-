# 6_FORM_PROJECT - Venda seu carro

## Descrição
Projeto de formulário de cadastro de veículos, com **HTML e CSS**.  
O objetivo é criar um formulário completo e organizado para que usuários possam cadastrar seus carros para venda.

O formulário permite inserir informações básicas do veículo, selecionar tipo de câmbio, marcar opcionais e fazer upload de múltiplas imagens.

---

## Arquivos do projeto

### 1. `index.html`
Estrutura do formulário:

- Inputs de texto (`<input type="text">`) para título, marca, modelo  
- Inputs numéricos (`<input type="number">`) para preço e kilometragem  
- Input de data (`<input type="date">`) para data de compra  
- Radio buttons para tipo de câmbio  
- Checkboxes para opcionais do carro  
- Upload de múltiplas imagens (`<input type="file" multiple>`)  
- Botão de envio (`submit`)  

### 2. `css/styles.css`
Estilização:

- Layout centralizado e limpo  
- Inputs com borda inferior e padding  
- Agrupamento visual de opcionais e campos  
- Botão de envio com efeito hover e cores destacadas  
- Backgrounds e cores suaves para leitura confortável  

---

## Tecnologias utilizadas
- HTML5  
- CSS3  

---

## Como rodar localmente
1. Clone este repositório ou faça download da pasta do projeto.  
2. Abra `index.html` no navegador.  
3. Preencha o formulário para testar os campos de texto, números, datas, radio buttons, checkboxes e upload de imagens.
