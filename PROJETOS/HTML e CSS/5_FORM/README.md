# 5_FORM - Formulários HTML

## Descrição
Projeto de prática de **formulários HTML**, utilizando diversos tipos de campos e atributos.  
O objetivo é entender como criar formulários completos e estilizados, com boa semântica, acessibilidade e organização visual usando **HTML5 e CSS3**.

O projeto inclui inputs, selects, textareas, radio buttons, checkboxes, datas, arquivos, emails, senhas e mais.

---

## Arquivos do projeto

### 1. `index.html`
- Estrutura do formulário:
  - Inputs de texto (`<input type="text">`)
  - Campos numéricos, emails, senhas e datas
  - Select simples e múltiplo (`<select>`)
  - Textarea para comentários
  - Fieldset com legend para agrupar campos
  - Datalist para sugestões de input
  - Radio buttons e checkboxes
  - Upload de arquivos
  - Botões de envio (`submit`) e reset
  - Uso de atributos: `required`, `placeholder`, `disabled`, `value`

### 2. `css/styles.css`
- Estilização básica:
  - Margem entre os elementos (`div { margin-bottom: 10px; }`)
  - Inputs e labels organizados em bloco (`display: block`)
  - Radio buttons e checkboxes alinhados horizontalmente (`display: inline-block`)

---

## Tecnologias utilizadas
- HTML5
- CSS3

---

## Como rodar localmente
1. Clone este repositório ou faça o download da pasta do projeto.  
2. Abra o arquivo `index.html` no seu navegador.  
3. Interaja com o formulário para testar os diferentes campos e atributos.

---

## Aprendizados
- Criação de formulários completos com **diversos tipos de input**  
- Uso de **atributos importantes**: `required`, `placeholder`, `value`, `disabled`  
- Estruturação e semântica de formulário: `<fieldset>`, `<legend>`, `<label>`  
- Criação de listas de opções com `<select>` e `<datalist>`  
- Agrupamento de opções com **radio buttons** e **checkboxes**  
- Boas práticas de estilo com CSS, alinhando labels e inputs corretamente

---

> **Sugestão de melhoria visual:**  
> Pode-se deixar o formulário mais moderno com:  
> - Bordas arredondadas e cores nos inputs  
> - Efeito hover nos botões  
> - Layout responsivo com Flexbox ou Grid  
> - Placeholder estilizado e fontes personalizadas
